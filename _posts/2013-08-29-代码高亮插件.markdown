正在使用的高亮插件Syntax Highlighter and Code Colorizer for WordPress
写文章的时候，在文本模式下将代码放在这个标签里：

    
    <pre class=”brush:code-alias"> your code </pre>


比如c++代码可以使用如下方式

    
    <pre class=”brush:c++"> your code </pre>
