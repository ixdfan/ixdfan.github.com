捕获到数据之后,我们想要找到符合特定条件的数据包,使用Ctrl+F打开Find Packet对话框.

对话框中有三种方式:

- Display filter
	可以通过数据表达式进行筛选.并且只找出符合表达式的数据包

- Hex value
	使用输入的十六进程对数据包进行搜索

- String
	使用输入的字符串对数据包进行搜索

	-------------------------------------------------------------------------------
	搜索类型				例子
	-------------------------------------------------------------------------------
	Display filter			not ip
							ip.addr==192.168.1.1
							arp
							ip
	-------------------------------------------------------------------------------
	Hex value               00:ff
	-------------------------------------------------------------------------------
	String                  passwd
	-------------------------------------------------------------------------------

到找到之后,你可以使用Ctrl+N向下查找下一个或是Ctrl+B向上查找上一个
